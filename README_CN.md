## 功能特性

- **🧠 36 种思维模型**：涵盖逻辑思考、创意思维、问题解决、市场营销、组织团队、经营战略六大领域
- **🌍 多语言支持**：完整的中英文界面，AI 根据语言设置进行相应语言的思考和回复
- **📊 思维导图可视化**：使用 Markmap 生成交互式思维导图
- **🔄 三步骤深度分析**：问题理解 → 深度分析 → 方案评估
- **🔐 本地数据存储**：隐私安全，数据不上传服务器
- **⚙️ 可自定义提示词**：用户可在设置中自定义所有 AI 系统提示词

## 快速开始

### 前置要求

- Node.js >= 18.0.0
- npm >= 9.0.0

### 安装

```bash
# 克隆仓库
git clone https://github.com/BigTooth/ideaverse.git
cd ideaverse

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 配置

1. 在应用设置页面配置您的 AI API Key
2. 支持 OpenAI API 和 DeepSeek API
3. 可自定义 API 端点地址

### 构建

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 思维模型

IdeaVerse 整合了 36 种经典思维模型，覆盖六大领域。每个模型都有特定的优势，适用于不同类型的问题。

### 逻辑思考能力

| 模型 | 说明 | 擅长解决的问题 |
|------|------|----------------|
| MECE 分解法 | 相互独立，完全穷尽 | 战略咨询、问题拆解、确保分析无遗漏 |
| 金字塔原理 | 结论先行、逻辑递进 | 沟通表达、报告撰写、清晰传达观点 |
| 六顶思考帽 | 六种思考视角激活多元思维 | 团队协作、创意讨论、避免思维盲区 |
| PREP 法 | 结论-理由-例证-结论 | 结构化表达、演讲汇报 |
| 归纳演绎法 | 归纳与演绎双管齐下 | 论证推理、逻辑验证 |

### 创意与想象力

| 模型 | 说明 | 擅长解决的问题 |
|------|------|----------------|
| 书面头脑风暴法 | 书面形式收集创意 | 团队创意、平等参与、避免权威主导 |
| KJ 法 | 卡片分类整合信息 | 信息整理、知识构建、发现隐藏关联 |
| 曼陀罗思考法 | 九宫格发散思维 | 创意发散、多维探索、突破思维定式 |
| SCAMPER 法 | 7维度激发创新 | 产品创新、流程优化、改进现有方案 |
| 利弊均衡表 | 优劣势对比分析 | 决策辅助、权衡利弊 |

### 问题解决

| 模型 | 说明 | 擅长解决的问题 |
|------|------|----------------|
| 5W2H 分析法 | 七维度全面分析 | 问题定义、信息收集、确保分析全面 |
| 逻辑树 | 逐层分解问题 | 复杂问题拆解、找出关键因素 |
| 空雨伞模型 | 事实→解读→行动 | 快速决策、行动导向 |
| 艾森豪威尔矩阵 | 重要度/紧急度分类 | 时间管理、任务优先级排序 |
| 假设思考 | 假设导向快速聚焦 | 问题诊断、快速验证 |
| 根因分析法 | 连续追问"为什么" | 问题溯源、找到根本原因 |
| 帕累托分析 | 80/20 法则 | 资源优化、识别关键少数因素 |
| 决策矩阵 | 多标准加权评分 | 选项比较、科学决策 |

### 市场营销战略

| 模型 | 说明 | 擅长解决的问题 |
|------|------|----------------|
| SWOT 分析 | 优势/劣势/机会/威胁 | 战略规划、全面评估现状 |
| PEST 分析 | 政治/经济/社会/技术 | 宏观环境分析、把握外部趋势 |
| 3C 分析 | 公司/顾客/竞争者 | 竞争战略、市场定位 |
| STP 分析 | 细分/目标/定位 | 市场定位、精准营销 |
| 波特五力分析 | 行业竞争强度评估 | 行业分析、竞争格局判断 |
| 市场营销组合 4P | 产品/价格/渠道/推广 | 营销策略制定 |
| AIDMA 模型 | 注意-兴趣-欲望-记忆-行动 | 消费者行为分析、营销漏斗优化 |
| 产品生命周期 | 导入/成长/成熟/衰退 | 产品策略、生命周期管理 |
| BCG 矩阵 | 市场增长率与份额分类 | 业务组合管理、资源配置 |

### 组织与团队

| 模型 | 说明 | 擅长解决的问题 |
|------|------|----------------|
| PDCA 循环 | 计划-执行-检查-处理 | 持续改进、质量管理 |
| 7S 模型 | 战略/结构/系统/人员/技能/风格/价值观 | 组织诊断、变革管理 |
| 马斯洛需求理论 | 需求层次激励 | 团队激励、员工管理 |
| 利益相关者分析 | 识别分析相关方利益 | 项目管理、变革管理、协调各方利益 |

### 经营战略

| 模型 | 说明 | 擅长解决的问题 |
|------|------|----------------|
| 安索夫矩阵 | 市场/产品组合增长战略 | 增长规划、业务拓展方向 |
| 波特三大战略 | 成本领先/差异化/集中化 | 竞争战略、建立竞争优势 |
| 价值链分析 | 价值创造活动分析 | 成本优化、差异化定位 |
| 情景规划法 | 多种未来情景应对 | 战略灵活性、不确定性管理 |
