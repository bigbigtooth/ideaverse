# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-16

### Added

- Initial open source release
- 36 scientific thinking models across 6 domains
- Multi-language support (Chinese and English)
- AI-powered deep analysis with streaming responses
- Interactive mind map visualization using Markmap
- Three-step analysis workflow: Problem Understanding → Deep Analysis → Solution Evaluation
- Local data storage for privacy protection
- Customizable AI system prompts
- Support for OpenAI and DeepSeek APIs
- Customizable API endpoint URL
- History management for analysis sessions
- Export functionality for analysis results

### Features

- **Thinking Models**: 36 models covering:
  - Logical Thinking: MECE, First Principles, Inductive/Deductive Reasoning, etc.
  - Creative Thinking: Six Thinking Hats, SCAMPER, TRIZ, etc.
  - Problem Solving: 5 Whys, Fishbone Diagram, Root Cause Analysis, etc.
  - Marketing: 4P Marketing, STP, AIDA, etc.
  - Team Organization: RACI Matrix, Team Canvas, etc.
  - Business Strategy: SWOT, Porter's Five Forces, Blue Ocean Strategy, etc.

### Technical

- Built with Vue 3.5 + Vite 7
- State management with Pinia 3
- Internationalization with Vue I18n 11
- AI integration with OpenAI SDK 6
- Mind map generation with Markmap 0.18
- Icon library: Lucide Vue Next
