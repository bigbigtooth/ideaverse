# 深度思界 IdeaVerse 产品说明书

## 产品概述

### 产品名称
**深度思界**（英文名：IdeaVerse）

### 品牌口号
**思至深处，界自成**（Deep thought, clear bounds）

### 产品定位
深度思界是一款 AI 驱动的深度思考助手，通过科学的思维模型和结构化的分析流程，帮助用户对复杂问题进行全方位、多维度的深度剖析，最终生成可执行的解决方案。

### 目标用户
- 企业管理者和决策者
- 咨询顾问和分析师
- 产品经理和战略规划人员
- 需要进行深度思考的个人用户

## 核心功能

### 一、三步骤深度思考流程

产品采用渐进式的三步骤思考流程，引导用户从问题理解到方案落地：

#### 步骤一：问题理解（Understanding）

**功能描述**  
通过 AI 智能采访的方式，帮助用户厘清问题的背景、边界和核心诉求。

**核心特性**
- AI 自动生成 3-5 个针对性采访问题
- 每个问题提供 3 个预设选项供快速选择
- 支持自定义回答，满足个性化需求
- 自动生成问题理解报告，包含：
  - 核心总结
  - 关键要点
  - 关注领域
  - 深层需求
  - 问题边界

**用户价值**
- 避免问题定义模糊导致的无效分析
- 挖掘用户潜在的真实需求
- 为后续分析奠定坚实基础

#### 步骤二：深度分析（Analysis）

**功能描述**  
运用经典思维模型对问题进行多维度拆解，生成深度分析报告。

**核心特性**
- AI 智能推荐 3 个最适合当前问题的思维模型
- 支持 36 种经典思维模型（详见思维模型库）
- 自动拆解 4-6 个分析维度
- 每个维度生成详细分析卡片，包含：
  - 现象描述
  - 原因分析
  - 影响评估
  - 隐藏因素
- 支持维度重新分析、编辑和删除
- 生成深度分析汇总报告

**用户价值**
- 系统化拆解复杂问题
- 避免单一视角的局限性
- 发现问题背后的深层逻辑

#### 步骤三：方案评估（Evaluation）

**功能描述**  
基于深度分析结果，生成创新解决方案并进行多维度评估。

**核心特性**
- 自动生成多个创新解决方案
- 三维度评估体系：
  - 有效性（0-10分）：能否解决问题
  - 可行性（0-10分）：资源/时间是否允许
  - 可持续性（0-10分）：长期效果如何
- 加权评分计算（有效性 50% + 可行性 30% + 可持续性 20%）
- 压力测试：最坏情况分析与应对措施
- 支持方案重新生成和编辑
- 生成思维导图可视化展示
- 支持导出 Markdown 格式

**用户价值**
- 获得可执行的解决方案
- 科学评估方案优劣
- 提前预判风险并准备应对

### 二、思维模型库

产品内置 36 种经典思维模型，覆盖六大领域：

#### 逻辑思考能力领域
| 模型 | 说明 | 适用场景 |
|------|------|----------|
| MECE 分解法 | 相互独立，完全穷尽 | 战略咨询、问题拆解 |
| 金字塔原理 | 结论先行、逻辑递进 | 沟通表达、报告撰写 |
| 六顶思考帽 | 六种思考视角激活多元思维 | 团队协作、创意讨论 |
| PREP 法 | 结论-理由-例证-结论 | 结构化表达 |
| 归纳演绎法 | 归纳与演绎双管齐下 | 论证推理 |

#### 创意与想象力领域
| 模型 | 说明 | 适用场景 |
|------|------|----------|
| 书面头脑风暴法 | 书面形式收集创意 | 团队创意、平等参与 |
| KJ 法 | 卡片分类整合信息 | 信息整理、知识构建 |
| 曼陀罗思考法 | 九宫格发散思维 | 创意发散、多维探索 |
| SCAMPER 法 | 7维度激发创新 | 产品创新、流程优化 |
| 利弊均衡表 | 优劣势对比分析 | 决策辅助 |

#### 问题解决领域
| 模型 | 说明 | 适用场景 |
|------|------|----------|
| 5W2H 分析法 | 七维度全面分析 | 问题定义、信息收集 |
| 逻辑树 | 逐层分解问题 | 复杂问题拆解 |
| 空雨伞模型 | 事实→解读→行动 | 快速决策 |
| 艾森豪威尔矩阵 | 重要度/紧急度分类 | 时间管理、任务优先级 |
| 假设思考 | 假设导向快速聚焦 | 问题诊断 |
| 根因分析法 | 连续追问"为什么" | 问题溯源 |
| 帕累托分析 | 80/20 法则 | 资源优化、关键因素识别 |
| 决策矩阵 | 多标准加权评分 | 选项比较 |

#### 市场营销战略领域
| 模型 | 说明 | 适用场景 |
|------|------|----------|
| SWOT 分析 | 优势/劣势/机会/威胁 | 战略规划 |
| PEST 分析 | 政治/经济/社会/技术 | 宏观环境分析 |
| 3C 分析 | 公司/顾客/竞争者 | 竞争战略 |
| STP 分析 | 细分/目标/定位 | 市场定位 |
| 波特五力分析 | 行业竞争强度评估 | 行业分析 |
| 市场营销组合 4P | 产品/价格/渠道/推广 | 营销策略 |
| AIDMA 模型 | 注意-兴趣-欲望-记忆-行动 | 消费者行为分析 |
| 产品生命周期 | 导入/成长/成熟/衰退 | 产品策略 |
| BCG 矩阵 | 市场增长率与份额分类 | 业务组合管理 |

#### 组织与团队领域
| 模型 | 说明 | 适用场景 |
|------|------|----------|
| PDCA 循环 | 计划-执行-检查-处理 | 持续改进 |
| 7S 模型 | 战略/结构/系统/人员/技能/风格/价值观 | 组织诊断 |
| 马斯洛需求理论 | 需求层次激励 | 团队激励 |
| 利益相关者分析 | 识别分析相关方利益 | 项目管理、变革管理 |

#### 经营战略领域
| 模型 | 说明 | 适用场景 |
|------|------|----------|
| 安索夫矩阵 | 市场/产品组合增长战略 | 增长规划 |
| 波特三大战略 | 成本领先/差异化/集中化 | 竞争战略 |
| 价值链分析 | 价值创造活动分析 | 成本优化、差异化 |
| 情景规划法 | 多种未来情景应对 | 战略灵活性 |

### 三、历史记录管理

**功能描述**  
自动保存所有思考会话，支持随时继续和回顾。

**核心特性**
- 自动保存思考进度
- 会话状态标识（进行中/已完成）
- 支持继续未完成的思考
- 支持导出会话数据（JSON 格式）
- 支持导出思维导图（Markdown 格式）
- 支持删除历史记录

### 四、设置中心

#### API 配置
- 支持配置 OpenAI 兼容的 API 服务
- 可自定义 Base URL（默认：DeepSeek API）
- 可自定义模型名称
- API Key 安全存储（本地浏览器）

#### 语言设置
- 支持简体中文（zh-CN）
- 支持英文（en-US）
- 实时切换，全局生效

#### AI 提示词自定义
- 支持自定义所有 AI 提示词
- 支持变量占位符
- 支持重置单个/全部提示词

#### 数据管理
- 本地数据存储
- 支持清除所有数据

## 产品特色

### 1. 科学方法论驱动
- 整合 36 种经典思维模型
- 每个模型都有明确的适用场景和优势说明
- AI 智能推荐最适合的模型组合

### 2. 结构化思考流程
- 三步骤渐进式引导
- 每一步都有明确的输入输出
- 避免思维跳跃和遗漏

### 3. 深度而非广度
- 专注于问题的深度剖析
- 区分相关性和因果性
- 挖掘现象背后的本质

### 4. 可视化输出
- 思维导图展示完整思考过程
- 支持多种格式导出
- 便于分享和存档

### 5. 隐私安全
- 所有数据本地存储
- API Key 仅存储在用户浏览器
- 不上传任何用户数据到服务器
